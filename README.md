# JobLogger

JobLogger is a full-stack web application developed for job seekers to keep track of their job applications, subscriptions, and related notes. It allows users to manage their job applications efficiently, add notes for each job, and update the application status. Users can also archive or delete jobs based on their preferences. The application is built using React.js, Node.js, Express, MongoDB, and Redux technologies.

## Features

- **Job Tracking:** Users can add and manage job applications, including details such as company name, position, and application status.
- **Notes:** Users can add notes for each job application to keep track of important details, interview dates, and follow-up actions.
- **Status Update:** Users can update the status of their job applications, such as "Applied," "Interview Scheduled," or "Offer Received."
- **Archiving and Deletion:** Users can archive completed applications or delete unwanted ones to keep their dashboard organized.
- **CV File Management (Upcoming):** Users will be able to upload and manage their CV files, download them, and attach them to specific job applications.
- **Custom Points (Upcoming):** Users will have the option to add custom points or tags to each job application for better categorization and filtering.

## Technologies Used

- **Frontend:**
  - React.js
  - Redux (State Management)
  - HTML5/CSS3 (Bootstrap for styling)
  - Axios (HTTP requests)

- **Backend:**
  - Node.js
  - Express (RESTful API)
  - MongoDB (Database)
  - Mongoose (ODM for MongoDB)

## Getting Started

To get started with JobLogger, follow these steps:

### Installation

1. Clone the repository to your local machine


2. Change directory to the project folder:

cd client
npm install
cd ../server
npm install


### Usage

1. Start the backend server:
cd server
npm start


The server will run on `http://localhost:5000`.

2. Start the frontend development server:

cd client
npm start


The React development server will run on `http://localhost:3000`.

3. Open your browser and visit `http://localhost:3000` to access the JobLogger web application.
